# Revia-Race-HSK
雷维亚HSK个人修改版，修复飞刀无法制作的bug，修改雷维亚霰弹枪属性使其能使用所有12号霰弹，给雷维亚添加了两种专属弹药和两把使用专属弹药的强力远程武器<br/>
血赐弩，等价于手枪，射速慢于手枪，但一次性能射出3发弹药（实际上就是把连射cd设置为0）<br/>
